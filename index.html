Helpful tips

1: Use the Block Shapes button to add different Blocks to your Art Board

2: Pick a Color and use it to re color Block Elements

3: Use the Rotate button to Rotate Block Sections

4: When you are done Hit the save button to save out your Creation as an image

5: Use the Menu Button at the top left to choose a new art board size and Start Again
